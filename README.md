#mamiline

開発者の所属組織が変わったため、forkしました。

"mamiline(マミライン)"は、Moodle上のデータのみでポートフォリオを作成するMoodleプラグインです。

詳細情報は下記ページに記載しています。
http://version2-inc.github.io/moodle-block_mamiline
